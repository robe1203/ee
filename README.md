# 📊 Sistema Web Contable PWA

Este proyecto consiste en el desarrollo de un sistema web contable orientado al ámbito académico, construido con Laravel, Vue.js e Inertia. La aplicación permite registrar, consultar y gestionar operaciones contables de manera sencilla y clara.

Además, el sistema ha sido diseñado como una **Aplicación Web Progresiva (PWA)**, lo que permite su uso tanto con conexión a internet como sin ella, facilitando el acceso desde distintos dispositivos.

---

## 🚀 Características principales

- Registro de pólizas contables
- Consulta de movimientos financieros
- Gestión de empresas
- Catálogo de cuentas
- Generación de reportes básicos
- Interfaz responsiva adaptable a dispositivos móviles
- Funcionamiento offline mediante almacenamiento local
- Sincronización de datos cuando se recupera la conexión

---

## 📱 Funcionalidad PWA

La aplicación incluye características propias de una PWA:

- Instalación en dispositivos móviles y de escritorio
- Uso sin conexión a internet
- Almacenamiento local de datos
- Sincronización automática al volver a estar en línea
- Cacheo de recursos para mejorar el rendimiento


---

## ⚙️ Tecnologías utilizadas

- Laravel (Backend)
- Vue.js (Frontend)
- Inertia.js (Comunicación entre frontend y backend)
- IndexedDB (Almacenamiento local offline)
- Service Worker (Cache y funcionamiento offline)
- Vite (Compilación de recursos)
